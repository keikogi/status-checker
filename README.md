Keikogi Status Checker
======================

Requirements
------------
PHP 5.3+

Installation
------------
Add this to a composer.json file:
```
"repositories": [
    {
        "type": "vcs",
        "url": "https://github.com/keikogi/status-checker"
    }
],
"require": {
    "keikogi/status-checker": ">=1.0.0"
}
```
